Key User Journey Flows
Critical paths through the platform for each user type

New Visitor → Booking

Public → Registered User
1
Land on luxury homepage, watch hero video
2
Browse Destinations (Kenya / Tanzania)
3
View Destination Detail + related packages
4
Select Package → view itinerary, pricing
5
Click "Book Now" → prompted to register/sign in
6
Complete booking + add gear bundle at checkout
7
Receive confirmation + account created

Returning User → Shop Purchase

Registered User Flow
1
Sign in → redirected to personal dashboard
2
See notification: "Gear for your Kenya trip"
3
Click through to Shop → pre-filtered by trip
4
Add safari jacket + binoculars to cart
5
Apply loyalty points discount at cart
6
Select currency (EUR) → pay via card
7
Order ships, tracking email sent

Sub-Admin → Add New Package

Sub-Admin Workflow
1
Log into admin portal → view dashboard KPIs
2
Navigate to Packages Manager
3
Click "Create New Package"
4
Fill itinerary, pricing, inclusions, gallery
5
Assign related gear products (cross-sell)
6
Set availability calendar + seasonal pricing
7
Publish → live on customer-facing site

Super Admin → Create Sub-Admin

Super Admin Workflow
1
Log in → Master Dashboard overview
2
Navigate to User Management
3
Click "Invite New User"
4
Enter email → assign role: Sub-Admin
5
Select permission modules (e.g., Kenya only)
6
Send invite → new admin sets password
7
Action logged in Audit Trail

Bundle Purchase Flow

Cross-sell: Tour + Gear
1
User views package detail page
2
"Complete Your Safari" section shows gear
3
Add bundle to cart with one click
4
Cart shows tour + gear with bundle saving
5
Checkout: select delivery to home or camp
6
Multi-currency payment processed
7
Single confirmation for both bookings

Sub-Admin → Manage Inventory

Inventory Alert Response
1
Dashboard shows low-stock alert badge
2
Navigate to Inventory Manager
3
View items below threshold (red indicators)
4
Click item → update stock quantity
5
Or raise restock request → notify supplier
6
Set alert threshold per product
7
Inventory report exported for review
